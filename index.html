Tutorial de FlexBox

O que é FlexBox ?

Flexbox é uma propriedade que usamos no CSS que nos permite fazer layouts com mais facilidade.


Com simples e poucos comandos conseguimos fazer layouts bonitos e intuitivos.


Assim como mostra a imagem ao lado veremos como organizar nosso conteúdo, tudo sem envolver muitas linhas de código.


Vamos aprender as seguintes propriedados do Flexbox:

  • align-itens
  • justify-content
  • flex-direction
  • flex-grow
  • flex-shrink

Como funciona o flexbox ?

O flexbox alinha todos os elementos filhos que estão abaixo dele.


Os elementos que estão abaixo tem uma relação de hierarquia, ou seja, o elemento que está abaixo é filho do elemento que está em cima e o elemento que está em cima é pai do que está abaixo. Caso tenha dúvidas observe a imagem ao lado.


Descrição:

A primeira div está encapsulando (encapsular é ter os elementos dentro - os elementos que estão dentro da div pai estão encapsulados dentro dela) os elementos filhos.

A div filho 1 e a div filho 2 são filhas da div pai porque estão abaixo dela.

Como podemos alinhar nosso layout com o flex-direction ?

Bem, passado algumas explicações de como o layout funciona e qual a relação de parentesco dos elementos vamos ver como alinhar os elementos com flexbox.


Existem dois tipos de alinhamentos mais comuns:

  • row
  • column

E como usar no CSS ?

É bem simples, no elemento pai basta colocar as propriedades display e flex-direction.
